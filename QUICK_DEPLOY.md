# 🚀 QUICK START - Deployment in 5 Minutes

## You Need:
1. GitHub account (free at github.com)
2. Git installed (1 min from git-scm.com)
3. A Vercel account (free with GitHub)
4. A Render account (free with GitHub)

---

## The Commands (Copy & Paste)

### In PowerShell (your project folder):

```bash
# First time setup
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# Initialize and push to GitHub
git init
git add .
git commit -m "Initial commit: SecureLedger app"
git remote add origin https://github.com/YOUR-USERNAME/SecureLedger.git
git branch -M main
git push -u origin main
```
